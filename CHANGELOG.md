## [1.0.0] - 2026-01-14

### 新增

#### 核心功能
- ✨ 实现完整的 RAG（检索增强生成）问答系统
- 🔍 基于向量相似度的智能文档检索
- 🎯 集成 Rerank 模型进行精确重排
- 🧠 集成 LLM 生成自然语言回答
- 📚 支持 Markdown 文档处理和分块

#### 技术实现
- 🚀 使用 FastAPI 构建高性能 Web 服务

- 🌐 提供 RESTful API 接口

### 技术细节

#### 模块结构
```
core/               # 核心功能模块
├── embeddings.py   # 嵌入引擎
├── vector_store.py # 向量存储管理
├── llm_client.py   # LLM 客户端
├── reranker.py     # 重排引擎
└── processor.py    # 文档处理器

utils/              # 工具模块
├── logger.py       # 日志管理
├── exceptions.py   # 异常定义
└── responses.py    # 响应模型
```

#### API 端点
- `POST /query` - 问答查询接口
- `GET /health` - 健康检查接口
- `GET /stats` - 统计信息接口
- `GET /docs` - API 文档（自动生成）

#### 配置参数
- 向量检索参数：召回数、阈值
- 重排参数：Top-K、分数阈值
- LLM 参数：模型 ID、温度
- 服务参数：主机、端口、日志级别
